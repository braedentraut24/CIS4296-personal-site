School Projects

Things I've done in class.
ShadowSkip

ShadowSkip was a final program created for the Mobile Sensing and Devices course. It allows users to wave a hand over their phone's proximity sensor to pause or play music. The app only allows for a wave to trigger a play or pause when the phone is face up and not moving or rotating for five seconds, at which point it becomes "enabled" and allows the user to use it to play or pause music. This was meant to ensure that no waves were accidentally triggered while the phone was in a user's pocket or actively being used. Overall it was a fun project because I was able to walk through all stages of planning and development with only one partner.

A Browser

This is the largest project I've done in Android. It is a simple browser that I made as a final assignment for Introduction to Mobile Application Development. It contains multiple fragments, classes, and views that allow someone to browse the internet with multiple pages open and keep bookmarks. It also has support for both landscape and portrait orientations. Overall this was a super challenging project as I had to really understand the Android SDK and it's inner workings to get this done per the instructors guidelines.

Sandwich Ordering Form

This was a very recent project for Server-Side Web Application Development that was a basic introduction to posting data from an HTML page to an ASPX page with C# CodeBehind. I created custom classes in C# for a Sandwich object that can calculate it's own price and return it's attributes. This was used to populate data fields in the ASPX page as an "order confirmation". Overall it was a great introduction to server-side development and the intricacies of the ASP.NET framework.

PDF to Excel Automation

Another project I completed during the summer of 2021 was an OCR powered PDF to Excel conversion project. The goal of this bot was to take PDF order confirmations and enter that data into an Excel spreadsheet, something which was normally done by a human. I was tasked with making this possible when our original pipeline used lead to a failed to deploy due to network restrictions. I transferred and translated local-based Python scripts into AWS Lambda where I set up a serverless pipeline that first ran AWS Textract on a document and converted it into a JSON file, turned that JSON file into an Excel sheet, and finally deleted scan artifacts like JSON files and PDF files. Overall I really was able to show my ability to work under pressure and a hard deadline while experimenting with technologies new to our team. The pipeline I created was used as a template for further OCR based projects. This project ended up saving more than $100,000 worth of employee time.
